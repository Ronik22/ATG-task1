
# ATG_task1

Public Chat Room

## Installation

Install via git

```bash
    $ git clone https://github.com/Ronik22/ATG-task1.git
    $ cd ATG-task1
    $ python -m venv venv
    $ source venv/Scripts/activate
    $ cd ../..
    (venv) pip install -r requirements.txt
    (venv) python manage.py runserver
```
